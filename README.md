# RealSense Viewer in Python

## Environment

    * Windows 10
    * Python 3.7.9
    * pyrealsense2 2.41.0.2666
    * opencv-python 4.4.0.46
    * numpy 1.19.3

## Usage

    git clone https://github.com/y-labo/RealSenseViewer.git
    cd RealSenseViewer
    pipenv sync
    pipenv shell
    python rs_viewer.py
